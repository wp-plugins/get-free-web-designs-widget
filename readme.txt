== Description ==

A widget that displays a nice looking feed of the most recent designs from
<a href="http://www.getfreewebdesigns.com">Get Free Web Designs</a>.  It's a
great way to spruce up your site and support a great community supported site in
the process.  Requires PHP5.

== Installation ==

1. Verify that you have PHP5, which is required for this plugin.
1. Upload the whole `get-free-web-designs-widget` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the widget to any widgetized area of your site (like your sidebar) through the widgets admin page

== Frequently Asked Questions ==

= Can I have use than one instance of this widget? =

Yes, the Get Free Web Designs Widget employs the multi-widget pattern, which allows you to not only have more than one instance of this widget on your site, but even allows more than one instance of this widget in a single sidebar.

= Can I follow more than one feed? =

Currently there is only one designs feed avaialble.  Get Free Web Designs plans to add more options in the future.

= Can I display only the designs I have submitted? =

Currently there is only one "all inclusive" designs feed avaialble.  Get Free Web Designs plans to add more options in the future.
